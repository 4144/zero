# Zero

Fuse filesystem backed by cloud storage and transparent local persistent cache


## Setup

Create a config.yml:
```
accountId: [...]
applicationKey: [...]
bucketId: [...]
sqliteFileLocation: [...]
```
and save in `~/.config/zero/`
